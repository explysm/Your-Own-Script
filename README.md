# Installation:
---
YOS Comes in a 7zip archive that includes:
               |_Your Own Script (application)
               |_Template Program.cs
               |_Your Own Script.csproj
               |_obj/

**Note:** The (Soon to be)Android version will only include the base IDE and C# compiler.
