The OS of Creation

Within this grimoire lies the knowledge of **YOScript**, a simple yet powerful language for shaping logic and command. Wield the **YOIDE** as your terminal, and master the arcane spells of code to bend reality to your will.

Artifact of Logic: YOScript

YOScript is an ancient, interpreted language, simple in form but limitless in application. It's built on clear, keyword-based commands, making it a perfect first step on your journey to becoming a master scripter. Each line is a spell; each command is a gesture of power. 

vs message = "Hello, Adventurer!"
print message

The YOIDE: Command Nexus

The **YOIDE** is your terminal, your direct link to the YOScript engine. It's where you write your incantations, see the results of your spells in the console, and manage the magical artifacts (your scripts). The editor is where you craft, the console is where you witness the magic unfold.

The Codex of Commands

These are the fundamental spells in your arsenal. Wield them wisely to bring your creations to life.

print

Displays a message or value in the console, a crucial spell for debugging and communication.

print "Hello"
print 10 + 5

input:

Prompts the user for input, allowing your script to gather information from the outside world.

vs name = ""
input: name

func: & call

Creates a reusable block of code (`func`) and executes it by name (`call`).

func: greet
    print "Hi there!"
call greet

list, add, get

Conjures a collection of items (`list`), adds to it (`add`), and retrieves a specific item (`get`).

list my_list
add my_list "apple"
vs fruit = ""
get fruit my_list 0

random, len

Generates a random number (`random`) or determines the length of a word or list (`len`).

vi num = 0
random num 1 10
vs word = "wiki"
len word

timer:

A time-dilation spell that pauses execution for a specified number of milliseconds.

print "Waiting..."
timer: 2000
print "Done!"

The Scribe's Memory: Variables

Variables are the enchanted scrolls on which you record data. Each type holds a different kind of information.

vs (String)

Stores text-based knowledge, like a phrase or a name.

vs name = "Yolo"
vs greeting = "Hi " + name

vi (Integer)

Stores numerical values, from a creature's health to a spell's cooldown.

vi age = 30
vi year = age * 2

vb (Boolean)

Stores a simple state: `true` or `false`, like whether a door is open or a quest is complete.

vb is_valid = true
vb is_ready = 10 > 5

The Threads of Control: Logic & Loops

These commands dictate the path your script will follow, allowing for decisions and repetition.

if: else if: else

Creates a branching path, executing different blocks of code based on a condition.

vi num = 10
if: num > 10
    print "Greater"
else if: num == 10
    print "Equal"
else
    print "Smaller"

loop:, break, continue

Repeats a block of code as long as a condition is met. Use `break` to exit early or `continue` to skip to the next iteration.

vi counter = 0
loop: counter < 5
    print "Looping..."
    counter = counter + 1
    if: counter == 3
        print "Skipping next print"
        continue
    print counter
